# 🍕 Bosch Pizza

> *sometimes i feel like pizza solves more problems than people do*

Um projetinho que começou simples… e foi ganhando vida aos poucos.
Esse é o **Bosch Pizza**, um sistema de gerenciamento de pizzas, clientes e usuários — com uma estética que mistura **vidro (glassmorphism)**, tons suaves e aquele toque quentinho de laranja 🍊

---

## ✨ Sobre o projeto

O Bosch Pizza é uma aplicação fullstack onde você pode:

* 🍕 Cadastrar e gerenciar pizzas (com preço!)
* 👤 Gerenciar clientes
* 🔐 Autenticação de usuários
* 👑 Controle de acesso (admin)
* 📊 Visualizar informações no dashboard

Tudo isso com uma interface moderna, leve e com efeito de vidro 💫

---

## 🎨 Interface

A estética do projeto segue um estilo:

* 🧊 Glassmorphism (blur + transparência)
* 🍊 Destaques em laranja (ações importantes)
* 🌙 Tema escuro elegante
* ✨ Interações suaves (hover, foco, animações)

Porque sim… CRUD também pode ser bonito.
